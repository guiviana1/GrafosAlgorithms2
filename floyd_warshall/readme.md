# Descrição:

Código que implementa o algoritmo de Floyd Warshall para cálculo de caminhos mínimos entre vértices.

# Compilação:

Para compilar o código, certifique-se de ter a biblioteca Boost Graph instalada.

Use o comando:

g++ -std=c++17 floyd_warshall.cpp -o floyd_warshall -lboost_graph -lboost_regex
